Task 2

Automation Testing

Dua skrip Bash yang dijadwalkan via cron: pengumpulan data CSV secara periodik dan pembersihan otomatis file yang sudah lebih dari sebulan.

collect_data.sh
Dijadwalkan 08:00, 12:00, 15:00 WIB
  • Panggil GET /api/data dari backend
  • Konversi JSON ke CSV via Python
  • Simpan ke /home/cron/
  • Log setiap eksekusi
clean_data.sh
Dijadwalkan setiap hari 00:00 WIB
  • Scan semua cron_*.csv di path
  • Hapus file dengan mtime > 30 hari
  • File lain tidak tersentuh
  • Log nama file dan ukuran yang dihapus
task2-automation / collect_data.sh — crontab entry
# Cron entries — tambahkan via: crontab -e # collect_data: 08:00, 12:00, 15:00 WIB (UTC+7 → UTC 01:00, 05:00, 08:00) 0 1,5,8 * * * /home/cron/collect_data.sh >> /home/cron/collect.log 2>&1 # clean_data: 00:00 WIB setiap hari (UTC 17:00) 0 17 * * * /home/cron/clean_data.sh >> /home/cron/clean.log 2>&1
Format nama file output: cron_DDMMYYYY_HH.MM.csv  —  Contoh: cron_19122024_15.00.csv

Data Processing — SQL

Lima query SQL untuk mengelola tabel employee: insert, update bulk, agregasi, sorting, dan subquery filtering.

01
INSERT — Tambah Employee Baru
Albert, Engineer, 24 Jan 2024, 2.5 year exp, salary $50
SQL
INSERT INTO employee (name, position, join_date, year_of_exp, salary) VALUES ('Albert', 'Engineer', '2024-01-24', 2.5, 50.00);
02
UPDATE — Salary Semua Engineer
Aaron, Allen, Albert diupdate ke $85
SQL
UPDATE employee SET salary = 85.00 WHERE LOWER(position) = 'engineer';
03
SUM — Total Pengeluaran Salary Tahun 2021
Karyawan aktif: join ≤ 31-Des-2021 dan release ≥ 1-Jan-2021 (atau masih aktif)
SQL
SELECT SUM(salary) AS total_salary_2021 FROM employee WHERE join_date <= '2021-12-31' AND (release_date IS NULL OR release_date >= '2021-01-01');
Aktif di 2021: Jacky + John + Alano + Aaron + Peter  =  $645
04
ORDER BY — TOP 3 Year of Experience
Descending, ambil 3 teratas
SQL
SELECT name, position, year_of_exp, salary FROM employee ORDER BY year_of_exp DESC LIMIT 3;
#1  Alano (Manager)14 yr
#2  John (Asst. Manager)12 yr
#3  Jacky (Solution Architect)8 yr
05
SUBQUERY — Engineer dengan Experience ≤ 3 Tahun
Menggunakan WHERE id IN (subquery)
SQL
SELECT id, name, position, year_of_exp, salary FROM employee WHERE id IN ( SELECT id FROM employee WHERE LOWER(position) = 'engineer' AND year_of_exp <= 3.0 ) ORDER BY year_of_exp ASC;
Masuk: Aaron (1 yr), Albert (2.5 yr)  —  Dikecualikan: Allen (4 yr)
